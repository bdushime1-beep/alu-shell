IO Redirections and Filters



This folder contains solutions for the "Introduction to Linux and IT Tools" project tasks 0..26. Each script is a two-line `#!/bin/bash` script that performs the required operation using standard commands (`cat`, `head`, `tail`, `cut`, `tr`, `rev`, `sort`, `uniq`, `grep`, `find`, `paste`, etc.).



Run each script from its directory as described in the project instructions.
