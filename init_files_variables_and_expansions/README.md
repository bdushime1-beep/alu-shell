# Shell, Init Files, Variables and Expansions

Shell scripting project covering init files, variables, expansions, and shell arithmetic on Ubuntu 20.04 LTS.

## Scripts

| File | Description |
|------|-------------|
| `0-alias` | Creates an alias `ls` with value `rm *` |
| `1-hello_you` | Prints `hello` followed by the current user |
| `2-path` | Adds `/action` to the end of `PATH` |
| `3-paths` | Counts the number of directories in `PATH` |
| `4-global_variables` | Lists all environment variables |
| `5-local_variables` | Lists all local variables, env variables, and functions |
| `6-create_local_variable` | Creates local variable `BEST="School"` |
| `7-create_global_variable` | Creates global variable `BEST="School"` |
| `8-true_knowledge` | Prints `128 + TRUEKNOWLEDGE` |
| `9-divide_and_rule` | Prints `POWER / DIVIDE` |
| `10-love_exponent_breath` | Prints `BREATH ** LOVE` |
| `11-binary_to_decimal` | Converts `BINARY` from base 2 to base 10 |
| `12-combinations` | Prints all two-letter combos (a-z) except `oo` |
| `13-print_float` | Prints `NUM` with two decimal places |
| `14-decimal_to_hexadecimal` | Converts `DECIMAL` from base 10 to base 16 |
| `15-rot13` | Encodes/decodes text using ROT13 |
| `16-odd` | Prints every other line from input, starting with first |
| `17-water_and_stir` | Adds WATER (base water) + STIR (base stir.), prints in base behlnort |
